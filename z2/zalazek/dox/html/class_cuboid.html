Opis szczegółowy

Jest to abstrakcyjna klasa bazowa dla modeli prostopadłościanów reprezentujących przeszkody. W definicji klasy zawarte są jedynie podstawowe elementu interfejsu tej klasy.

Zakłada się, że podstawowymi atrybutami, których wartości mają być udostępniane, to położenie środka prostopadłościanu oraz długość poszczególnych boków.

virtual const Wektor3D& Cuboid::GetSize ( )
inlinevirtual

Udostępnia długości boków prostopadłościanu. Zakłada się, że prostopadłościany są tak ustawione, że jego ścianki są równoległe do jednej z powierzchni OXY, OXZ lub OYZ. Każda współrzędna wektora zawiera długość boku, który jest równoległy do danej osi. Tak więc np. współrzędna x-owa zawiera długość boku, który równoległy jest do osi OX.
